# СРСП №3. Основы программирования на JavaScript

Этот проект подготовлен как основа для сдачи СРСП №3 по теме «Основы программирования на JavaScript». В репозитории есть:

- локальная интерактивная викторина на HTML/CSS/JavaScript;
- готовый банк заданий для переноса в WordWall;
- исходные JavaScript-файлы с примерами для сложных заданий;
- структура, которую удобно опубликовать на GitHub и приложить ссылку в описании викторины.

## Состав проекта

- `index.html` — интерактивная викторина для локального запуска;
- `styles.css` — оформление интерфейса;
- `app.js` — логика викторины и список заданий;
- `wordwall/quiz-bank.md` — 16 готовых заданий для WordWall;
- `src/challenge-scripts.js` — скрипты и примеры, использованные в заданиях.

## Что покрывает викторина

- типы данных и операторы;
- условия и циклы;
- функции и объекты;
- DOM и события;
- массивы и методы `map`, `filter`, `forEach`;
- задания на отладку и исправление ошибок;
- задания на определение вывода в консоль.

## Как открыть локальную версию

1. Откройте файл `index.html` в браузере.
2. Ответьте на 16 вопросов.
3. После завершения будет показан итоговый балл и краткая аналитика по темам.

## Как перенести в WordWall

1. Откройте файл `wordwall/quiz-bank.md`.
2. Возьмите пары из раздела `Match` для формата «Сопоставление».
3. Возьмите вопросы из раздела `Open the Box` для формата «Open the Box».
4. В описание викторины вставьте ссылку на ваш GitHub-репозиторий.

Пример ссылки:

`https://github.com/Fomar07/-3-3-`

## Что рекомендуется приложить при сдаче

- ссылку на WordWall-викторину;
- ссылку на GitHub-репозиторий;
- 1-2 скриншота локальной викторины или кода из `src/challenge-scripts.js`, если преподаватель просит подтверждение авторства.

## Примечание

Если будете публиковать проект в GitHub, замените шаблон ссылки на вашу реальную ссылку на репозиторий и при необходимости обновите описание викторины под стиль вашего преподавателя.
