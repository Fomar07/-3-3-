# Шаблон описания для WordWall

Интерактивная викторина по теме «Основы программирования на JavaScript». Задания охватывают типы данных, операторы, условия, циклы, функции, объекты, массивы, методы `map`, `filter`, `forEach`, а также основы работы с DOM и событиями.

В викторине есть задания на:

- определение результата работы кода;
- исправление ошибок в коде;
- выбор правильного вывода в консоль;
- сопоставление JavaScript-конструкций и результата.

Исходные коды сложных заданий:

`https://github.com/Fomar07/WebSRSP3`

Формат выполнения: WordWall (`Match`, `Open the Box`).
