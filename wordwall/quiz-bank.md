# WordWall Quiz Bank

Ниже подготовлены 16 заданий для переноса в WordWall. Они уже покрывают все обязательные темы из СРСП №3.

## Match

Эти пары удобно перенести в шаблон WordWall `Match`:

| № | Левый элемент | Правый элемент |
|---|---|---|
| 1 | `typeof null` | `object` |
| 2 | `2 + '2'` | `22` |
| 3 | `5 === '5'` | `false` |
| 4 | `0` в условии `if` | falsy |
| 5 | `document.getElementById('title')` | поиск элемента по `id` |
| 6 | `button.addEventListener('click', handler)` | обработка клика |
| 7 | `[1, 2, 3].map(x => x * 2)` | `[2, 4, 6]` |
| 8 | `[1, 2, 3, 4].filter(x => x % 2 === 0)` | `[2, 4]` |
| 9 | `[1, 2, 3].forEach(item => total += item)` | сумма элементов |
| 10 | `document.querySelector('.card').textContent = 'Ready'` | изменить текст элемента |

## Open the Box

Эти задания удобно перенести в шаблон WordWall `Open the Box`.

### 1. Циклы

**Вопрос:** Чему будет равно `sum`?

```js
let sum = 0;
for (let i = 1; i <= 3; i++) {
  sum += i;
}
```

**Ответ:** `6`

### 2. Циклы

**Вопрос:** Какое финальное значение получит `count`?

```js
let count = 0;
while (count < 3) {
  count++;
}
```

**Ответ:** `3`

### 3. Отладка

**Вопрос:** Как исправить ошибку?

```js
const arr = [10, 20, 30];
for (let i = 0; i <= arr.length; i++) {
  console.log(arr[i]);
}
```

**Ответ:** Заменить `<=` на `<`

### 4. Функции

**Вопрос:** Что выведет код?

```js
function greet(name = 'JS') {
  return `Hello, ${name}!`;
}

console.log(greet());
```

**Ответ:** `Hello, JS!`

### 5. Объекты

**Вопрос:** Как правильно получить свойство `name` у объекта `user`?

```js
const user = { name: 'Aruzhan', age: 19 };
```

**Ответ:** `user.name`

### 6. Отладка

**Вопрос:** Что нужно исправить в условии?

```js
let a = 5;
if (a = 10) {
  console.log('OK');
}
```

**Ответ:** Заменить `=` на `===`
